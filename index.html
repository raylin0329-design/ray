歡迎來到我的學習歷程網頁。在高中期間，我學習了 Python 與 C++ 的基礎程式設計，並具備基本的程式邏輯與問題解決能力。

同時，我運用 GitHub 建立並管理自己的網站，累積實作經驗。

關於我

我是一位具有責任感的學生，在學習與生活中都能認真負責完成自己的任務。 在高中的學習歷程中，從高一到高三的學業成績皆維持穩定且良好，展現持續努力與自我要求的態度。

在校期間曾擔任優良學生，並參與商業活動相關事務，擔任商業際副負責人，培養了溝通協調與團隊合作的能力。

在興趣方面，我喜歡運動，例如打羽毛球與騎腳踏車，透過運動讓自己保持良好的體能與專注力。

我希望未來能持續精進資訊與程式相關能力，並將所學應用於實際生活與未來發展中。

專題

我們專題以烏龍茶蛋糕為核心商品，打造品牌「糕什麼烏龍」，結合茶文化與創新甜點，發展具市場差異化的產品。我們運用AI工具進行市場調查分析、視覺設計與行銷素材製作，提升企劃的效率與創意表現。

在企劃過程中，透過問卷分析了解消費者對茶類甜點的偏好，進一步明確品牌定位與目標客群，並規劃相應的行銷策略，如社群宣傳與視覺內容設計，使產品更具吸引力與市場競爭力。

本專題不僅展現我們將AI應用於行銷實務的能力，也培養了從產品發想到品牌規劃的整體企劃思維。
